//////////////////////////////////////////////////////////////////////////////////////////////////////////////////

Este proyecto cumple el propósito de dar préstamos y reservas de computadoras y salón multiusos
en la biblioteca de Administración de Empresas. Permite a los usuarios realizar préstamos de computadoras, 
agregar programas a las mismas, reservar salones y recibir computadoras al finalizar el préstamo.


//////////////////////////////////////////////////////////////////////////////////////////////////////////////////

Inicio de sesión:

Usuario: Admin

Contraseña: BibEmp


//////////////////////////////////////////////////////////////////////////////////////////////////////////////////
Menú principal:

Agregar, eliminar y revisar préstamos: la tabla muestra los préstamos que están corriendo hasta el momento, 
dando el ID del préstamo, el número de propiedad de la máquina que se prestó, el nombre y apellido de la persona 
que realizó el préstamo y la fecha a la cual se debe devolver la máquina.

Agregar tipo de usuario: se pueden agregar dos tipos de usuarios, profesor y estudiante. 
El profesor tiene una ID que se genera automáticamente, mientras que el estudiante requiere un número de estudiante.

Agregar programa: se pueden agregar programas a las computadoras, especificando el nombre del programa y el ID 
del computador al que se va a agregar. Cada programa debe estar ligado a un computador.

Agregar computadora: se pueden agregar computadoras de sistema o de Fire. Las computadoras de Fire tienen 
número de propiedad y barcos, mientras que las de sistema de biblioteca solamente tienen nombre de la propiedad.

Eliminar usuarios, programas y computadoras: se pueden eliminar completamente de la base de datos los usuarios, 
programas y computadoras que no estén relacionados con otras cosas.

Realizar préstamos: se pueden realizar préstamos de computadoras, especificando el número de la propiedad, l
a persona que la solicita y la fecha de inicio y fin del préstamo.

Recibir computadoras: se puede registrar la devolución de las computadoras prestadas al finalizar el préstamo.

Reservar salón: se puede reservar un salón para una actividad, especificando el nombre de la persona que 
realiza la reserva, su ID y la hora de inicio y fin de la actividad junto con la fecha.


//////////////////////////////////////////////////////////////////////////////////////////////////////////////////
Cómo usarlo:

Para agregar se debe seleccionar la opción correspondiente y llenar todos los campos requeridos en el
formulario que quieras usar, ya sea para agregar usuario, programa o computadora.

Para eliminar, hay que llenar el formulario con los datos de lo que se quiera eliminar ya sea para agregar usuario,
programa o computadora.

Para realizar un préstamo hay que llenar los campos requeridos en el formulario.
Una vez hecho esto, la información del préstamo se mostrará en la tabla principal de la pantalla.

Para recibir una computadora al finalizar el préstamo, hay que escanear el código de barras de la máquina.
La computadora recibida se quitará junto con el préstamo.

Se reserva el salón con las credenciales del usuario, se indica la fecha y la hora de inicio y culminación de la 
actividad.
